
## 推荐开发目录
一个好的开发目录是一个好的开始。作者推荐一个比较好的开发目录给大家。

```js
lib
├── api               // 网络请求，后端接口
├── assets            // 静态资源
├── components        // 自定义组件
├── containers        // 容器组件
├── layouts           // 布局组件
├── pages             // 页面
├── router            // 路由器
├── routes            // 路由表
├── storage           // 本地存储
├── stores            // 应用状态
├── theme             // 主题
├── utils             // 工具目录
└── main.dart         // 入口文件
```
